Core Study Topics

  • Patterns & Loops: A pattern is something that repeats in the same order (colors, shapes, sounds). A loop lets a computer repeat steps without being told every time.
  • ScratchJr Events: Events are triggers like clicking the Green Flag or "Start on Bump" that tell characters when to start moving or talking.
  • Digital Storytelling: Stories have three main parts: Beginning, Middle, and End. Breaking stories into "scenes" helps us organize them better.
  • ScratchJr Skills: We learned to add Sprites (characters), change backdrops, and send messages between characters to make a story flow.
  • Pattern Examples: Shapes, figures, colors, letters (A D A D), sounds, and actions (Stop, Squat, Jump, Jump).

Technical Glossary

Trigger / Event

A trigger is a signal that makes something happen. An event is the action that follows the trigger.

Bug & Debugging

A bug is a problem that stops a program. Debugging is the process of finding and fixing that problem.

Instructions

Steps or commands that show how to do something or make a character move.
