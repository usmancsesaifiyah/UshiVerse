Module 01: Data Transmission Logic

Website vs. Email: The HOTS Comparison

Both rely on IP Addresses and Routers to guide data. However, visiting a website is a "Pull" request (you ask the server to show you its content), whereas Email is a "Push" message through special servers to a private inbox.

// DNS & Packet Protocol
1. DNS: Resolves website name to IP Address.
2. Packets: Data is broken into small pieces for speed.
3. Routing: Routers direct packets via the fastest "clean" path.
4. Assembly: Receiver reorders packets using sequence numbers.

Note: We use sequence numbers because packets take different paths to avoid digital traffic jams.

Loop Structures

for loop:

Repeats a specific number of times. Perfect for drawing shapes (e.g., 4 sides of a square).

while loop:

Continues as long as a condition is True.

Python Syntax & Indentation

"Why does the space at the start of a line matter?"

Indentation tells Python which code belongs inside the loop. Without correct syntax and spacing, the program won't know when to stop or what to repeat.

HOTS: Loops simplify tasks by reducing repetitive commands into a single set of instructions, making code shorter, cleaner, and less prone to errors.
