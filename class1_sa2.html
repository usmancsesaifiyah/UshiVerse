Core Study Topics

  • Patterns & Loops: A pattern is something that repeats in the same order (colors, shapes, sounds). A loop lets a computer repeat steps without being told every time.
  • ScratchJr Events: Events are triggers like clicking the Green Flag or "Start on Bump" that tell characters when to start moving or talking.
  • Digital Storytelling: Stories have three main parts: Beginning, Middle, and End. Breaking stories into "scenes" helps us organize them better.
  • ScratchJr Skills: We learned to add Sprites (characters), change backdrops, and send messages between characters to make a story flow.
  • Pattern Examples: Shapes, figures, colors, letters (A D A D), sounds, and actions (Stop, Squat, Jump, Jump).
  • Pattern: A pattern is something that repeats again and again in the same order. Examples include colors (🔴🔵🔴🔵), letters (A D A D A D), and actions (Jump, Clap, Jump, Clap). Patterns help us predict what comes next.
  • Loop: A loop means doing the same steps again and again. In programming, loops save time because we do not need to write the same instruction many times.
  • Trigger & Event: A trigger is a signal that makes something happen. An event happens because of a trigger. Example: Clicking the Green Flag in ScratchJr starts actions.
  • Bug & Debugging: A bug is a mistake in a program. Debugging means finding and fixing the mistake so the program works correctly.
  • Digital Storytelling: Stories can be told using pictures, sounds, and digital tools. Every story has three main parts — Beginning, Middle, and End. ScratchJr helps us create digital stories using characters (sprites) and scenes (backdrops).

Technical Glossary - Grade 01

Loop

Doing the same steps again and again. In programming, a loop helps repeat steps without telling the computer every time. The Repeat block in ScratchJr helps repeat actions easily.

Trigger

A signal that makes something happen.

Event

Something that happens because of a trigger. Events like clicking the Green Flag or clicking on a character make things happen. Action is triggered when the Green Flag is clicked or when using the Start on Bump block. Events are important because they help make things happen in real life and on computers.

Bug

A problem in a program that stops it from working properly.

Debugging

Finding and fixing problems in a program.

Fix

To make something work again.

Instructions

Steps or commands that show how to do something or make a character move. Events are like instructions that tell characters what to do.

Digital Storytelling

Stories can be told using pictures, sounds, and digital tools. Every story has three main parts: Beginning, Middle, and End. Breaking stories into scenes helps us understand each part better. We used ScratchJr to create fun stories by adding sprites (characters), backdrops, movements, sounds, and messages between characters.
