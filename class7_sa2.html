Module 02: CSS Architecture (Syntax)

CSS controls layout, colors, and fonts across multiple pages efficiently.

// CSS Declaration Block
h1 {
  background-color: lightblue;
  color: darkblue;
  font-size: 30px;
}

HOTS: Why use CSS? Consistency (one file styles the whole site), Efficiency (updates are instant), and Responsiveness (works on mobile/tablets).

Module 03: The Development Lifecycle

Wireframing & Storyboarding

Wireframing is the blueprint of a webpage. Storyboarding uses frames and captions to visualize the flow before a single line of code is written.

Skipping Storyboards: Leads to wasted time, confused UI/UX, and expensive coding errors.
Designer

Creates the visual look, UI elements, and overall UX feel.

Front-End Dev

Uses HTML/CSS/JS to build the part users see and touch.

Back-End Dev

Works with servers and databases (the "hidden" logic).

Level 7 Glossary

Grade 7 Final Assessment

CSS Selector

Targets specific HTML elements (like p, h1, or div) to apply design rules.

Native vs Web App

Native: Built for one OS (iOS/Android). Web: Runs in a browser (Chrome/Safari).

HTML Tags

Special words in <brackets> like <h1> (headings) or <p> (paragraphs).

Logical Thinking

The systematic reasoning used to solve code bugs and engineering problems.
