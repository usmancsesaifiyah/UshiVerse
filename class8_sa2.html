BLK 09: Debugging & Loop Errors

Key Definitions

  • Debugging: Process of finding and fixing mistakes in code.
  • Syntax Error: Mistake that makes code unreadable to the computer.
  • Runtime Error: Error that happens while the program is running.
  • While Loop: Repeats as long as a condition is true.
  • Indentation Error: Happens when Python cannot identify code blocks due to wrong spacing.

Given Code Issues

Bugs in the code:

  • Missing indentation inside functions
  • Wrong operator: start - = 1
  • Incorrect quotation marks
  • Variable mismatch: suma vs sum
// Updated Correct Code
def count_down(start): while start > 0: print(start) start -= 1 def greet_user(name): print("Hello " + name) def add_numbers(a, b): suma = a + b print("The sum is:", suma) name = input("Enter your name: ") greet_user(name) num1 = int(input("Enter first number: ")) num2 = int(input("Enter second number: ")) add_numbers(num1, num2) count_down(5)

Infinite & Off-by-One Errors

  • Infinite loop happens when condition never changes.
  • Off-by-one error happens when loop runs one extra or one less time.
  • Fix by updating loop variable correctly.
  • Always check loop condition carefully.

BLK 10: Data, Patterns & Ethics

Key Terms

  • Data: Information collected for analysis.
  • Patterns: Trends found in organized data.
  • Ethical Decision-Making: Using data responsibly and respecting privacy.

Why Collect Data in a Structured Way?

  • Keeps data organized
  • Makes pattern identification easier
  • Saves time during analysis
  • Ensures consistency and accuracy
  • Supports better decision-making

Ethical Data Use

  • Respect privacy
  • Never share personal data without permission
  • Follow privacy laws
  • Build trust through responsible data handling

BLK 20: Cybersecurity & Online Safety

Common Cyber Threats

  • Phishing – Fake emails asking for personal data
  • Ransomware – Locks files and demands payment
  • Malware – Harmful software
  • Social Engineering – Manipulating people

Consequences

  • Stolen data
  • Financial loss
  • Device damage

Protection Methods

  • Use strong passwords
  • Enable 2FA
  • Avoid suspicious links
  • Install updates
  • Use antivirus software
